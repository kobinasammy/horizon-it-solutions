# Horizon IT Solutions

A full-stack web application built with Python Flask, deployed on Microsoft Azure.

Live site: https://horizonitsolutions-eabcdkbudrb9fuhb.austriaeast-01.azurewebsites.net 

Tech Stack
- Python / Flask
- PostgreSQL (Azure)
- SendGrid (transactional email)
- Azure App Service
- Custom CSS design system

Features
- 7-page responsive website
- Contact form with email notifications
- Job application form with CV upload
- SQLAlchemy ORM with PostgreSQL
